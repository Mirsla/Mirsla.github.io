发表于 2017-08-14 | 分类于 博客 | 阅读次数

GitHub的博客搭建

GitHub是一个很不错的开放平台，不仅有许多好的个人开源项目，还能托管静态页面等等。很早之前就想在GitHub上搭建自己的个人站，由于种种原因一直没有搭建。终于下定了决心，折腾了一天，终于在GitHub上吧博客搭建起来了，还是挺有意思的一件事情，所有就写一篇博客将怎样搭建博客记录下来，希望可以帮到更多的人。

网上的教程也有很多，但是我在搭建的过程中还是遇到了很多问题，现在将这些问题记录下来，如果有下一次的话也会方便很多，另外我也会在博客上不定期的更新关于怎么优化博客方面的东西

下面就开始介绍怎么一步一步的在GitHub上搭建个人博客

GitHub上的准备工作

申请GitHub帐号

帐号申请很简单，这里就不做过多的描述了，申请帐号请点这里

申请一个仓库

  • 登录GitHub后点击你的个人头像左侧的+号，选择New repository

  • 进去之后在Repository name种输入你的仓库名字，建议：使用username.github.io这样的形式，username请更换为你的用户名，比如我的用户名为Mirsla，那么你在Repository name种可以输入Mirsla.github.io

  • 点击Create repository这样你的个人仓库就建好了

GitHub pages

  • 进入你刚才建好的仓库，简历Github pages页面。在仓库页面点击Settings，往下拉，找到Github pages，如下图所示：

  • 进去之后随便选择一个主题点击确定就行了，因为最后使用Hexo，所以这里的主题不重要。
